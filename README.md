# 💰 مدبّر | Modabber

لوحة تحكم مالية حديثة تساعد المستخدم على إدارة أمواله بذكاء وتتبع الدخل والمصروفات والميزانيات والمدخرات في واجهة عربية عصرية.

## ✨ المميزات

* واجهة مستخدم عربية (RTL)
* تصميم حديث ومتجاوب (Responsive)
* عرض الرصيد الحالي والدخل والمصروفات
* تتبع المعاملات الأخيرة
* إدارة وسائل الدفع (Visa - Mastercard - PayPal)
* متابعة الفواتير الشهرية
* ملخص الحسابات البنكية
* نظرة عامة على الميزانية
* رسوم بيانية للدخل والمصروفات

## 🛠️ التقنيات المستخدمة

* HTML5
* CSS3
* Flexbox
* CSS Grid
